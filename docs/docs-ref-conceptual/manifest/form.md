# <a name="form-element"></a>Form 要素

メール アドインが特定のデバイス (デスクトップ、タブレット、または電話) で実行されているときに使用するフォームの UX の設定。

**アドインの種類:** メール

## <a name="syntax"></a>構文

```XML
<Form>
   ...
</Form>
```
